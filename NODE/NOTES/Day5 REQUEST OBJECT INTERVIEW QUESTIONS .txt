1Ô∏è‚É£ What is Response Object in Express?

The Response Object (res) represents the HTTP response that the Express server sends back to the client.

It is used to:

Send data to client

Send JSON

Send status codes

Send headers

End the request‚Äìresponse cycle

Example:

app.get("/", (req, res) => {
  res.send("Hello Client");
});


Here, res.send() sends response to client.

2Ô∏è‚É£ Different methods available in Response Object

Common methods:

Method	Description
res.send()	Send response (string, object, html)
res.json()	Send JSON response
res.status()	Set HTTP status code
res.sendFile()	Send file
res.redirect()	Redirect to another route
res.set()	Set header
res.header()	Set header
res.end()	End response
res.download()	Download file

Example:

res.status(200).json({message: "Success"});

3Ô∏è‚É£ How is res object made available? Do we import it?

No ‚ùå
We DO NOT import it manually.

Express automatically provides:

(req, res)


Example:

app.get("/", (req, res) => {
   res.send("Hello");
});


Express internally creates and passes the res object.

4Ô∏è‚É£ What kind of data can be sent using response object?

You can send:

String

JSON

HTML

File

Object

Buffer

Examples:

res.send("Hello");

res.json({name: "Abishek"});

res.sendFile("index.html");

5Ô∏è‚É£ Use of status codes in HTTP responses

Status codes tell client about result of request.

Example:

Code	Meaning
200	Success
201	Created
400	Bad Request
401	Unauthorized
403	Forbidden
404	Not Found
500	Server Error

Example:

res.status(404).send("Not Found");

6Ô∏è‚É£ What is a status code? Explain with examples

Status code is a number sent by server to indicate request result.

Examples:

Success:

res.status(200).send("OK");


Created:

res.status(201).send("User Created");


Error:

res.status(500).send("Server Error");


Forbidden:

res.status(403).send("Access Denied");
