2Ô∏è‚É£ Difference between Synchronous and Asynchronous functions
Synchronous (Blocking)

Executes line by line. Next line waits until previous finishes.

Example:

console.log("Start");
console.log("Middle");
console.log("End");

Output:

Start
Middle
End
Asynchronous (Non-blocking)

Does not wait. Runs in background.

Example:

console.log("Start");

setTimeout(()=>{
  console.log("Middle");
},2000);

console.log("End");

Output:

Start
End
Middle

Why important?

Prevents server blocking

Handles multiple users efficiently

5Ô∏è‚É£ What is Express and how is it different from Node HTTP module?
Definition

Express.js is a web framework for Node.js used to build APIs and servers easily.

Node HTTP module example (complex)
const http = require("http");

http.createServer((req,res)=>{
 res.end("Hello");
}).listen(3000);
Express example (simple)
const express = require("express");
const app = express();

app.get("/",(req,res)=>{
 res.send("Hello");
});

app.listen(3000);
Difference
Node HTTP	Express
Basic module	Framework
Hard to write	Easy
No routing support	Built-in routing
No middleware	Supports middleware
