Class Component

Uses ES6 class

Extends React.Component

Has lifecycle methods

Uses this.state and this.setState()

Functional Component

Uses JavaScript functions

Uses Hooks (useState, useEffect)

Simpler and preferred in modern React

Key Differences

Feature	Class Component	Functional Component
Syntax	Class-based	Function-based
State	this.state	useState
Lifecycle	Lifecycle methods	Hooks
this keyword	Required	Not needed
Performance	Slightly heavier	Lighter & faster
Usage today	Less preferred	Recommended
