2Ô∏è‚É£ What is Virtual DOM? How is it different from Real DOM?

Virtual DOM

A lightweight JavaScript copy of the Real DOM

React updates this first

Compares changes using diffing algorithm

Updates only the changed parts in Real DOM

Real DOM

Actual browser DOM

Direct manipulation is slow

Re-renders entire UI on changes

Difference
Virtual DOM	Real DOM
Fast updates	Slower
JS object	Browser structure
Updates selectively	Re-renders more
Better performance	Performance heavy
